受入保留アルゴリズムを用いたゼミマッチングのための shiny アプリ。所属先の業務で使用することを想定しています。

## Installation

GitHub レポジトリからインストールしてください。

``` r
# install.packages("remotes")
remotes::install_github("kenjisato/omuecon1a")
```

## プロジェクトディレクトリのセットアップ

``` r
omuecon1a::prep()
```


## アプリ起動

``` r
omuecon1a::up()
```
